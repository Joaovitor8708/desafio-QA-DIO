# 🚀 Desafio de QA - Documentação de Testes Manuais (Contexto ERP)

## 📌 Sobre o Projeto
Este repositório contém a entrega do desafio de projeto de testes manuais funcionais do bootcamp. 

O objetivo principal foi planejar e documentar o fluxo de trabalho de qualidade e o ciclo de vida de bugs (*Bug Lifecycle*), além de elaborar User Stories e Casos de Teste aplicados a um cenário real de 
**sistemas ERP**. As funcionalidades testadas envolvem o módulo de suporte ao cliente e validações de atualizações em lote no banco de dados.

## 🗂️ O que você vai encontrar na documentação (PDF)
A documentação completa está disponível no arquivo `.pdf` anexado neste repositório e inclui:

1. **Fluxos de Trabalho e Status:** Definição do board (To Do > In Progress > Ready for QA > In Test > Done).
2. **Ciclo de Vida do Bug:** Mapeamento desde o *New* até o *Closed*.
3. **User Stories (US):** - *US 01:* Modificação de Cadastro em Lote no Banco de Dados.
   - *US 02:* Alerta de Chamado Crítico no Painel de Suporte.
4. **Mind-Map:** Mapeamento visual das regras de negócio, exceções e cenários de sucesso da US 01.
5. **Casos de Teste (Step-by-Step):** Passos detalhados para validação de upload de planilhas CSV e manipulação de status.
6. **Casos de Teste (BDD):** Cenários descritos em Gherkin (Dado/Quando/Então) focados em permissões de acesso e alertas de interface.

## 🛠️ Ferramentas e Conceitos Aplicados
* **Gestão e Planejamento:** Jira, Confluence, Metodologias Ágeis (Scrum)
* **Design de Testes:** Testes Funcionais, Step-by-Step, BDD (Gherkin)
* **Diagramação:** Mind Maps (Mermaid/XMind)
* **Contexto de Domínio:** Sistemas de Gestão (ERP), Validação de Dados, Suporte Nível 2.

## 🚀 Próximos Passos
Como evolução natural deste projeto focado em testes manuais, o próximo passo seria a implementação de **Testes Automatizados**. 
O conhecimento prévio em linguagens como **Java** ou **PHP**, aliado a consultas estruturadas em **SQL** para validação direta no banco, permitiria automatizar 
os cenários descritos nas User Stories deste desafio.
